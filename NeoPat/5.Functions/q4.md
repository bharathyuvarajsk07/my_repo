Question No: 1
Single File Programming Question
Asha’s birthday is shortly coming and her parents have planned to arrange for a house party. Deepa was Asha’s best friend and was expecting her birthday for a month. This is because Deepa’s Dad has promised her that he and Deepa together would design a Reverse Talking kitty toy all by themselves and gift it to Asha. Deepa believed that Asha might be overjoyed with this gift from her dear friend.



Deepa’s Dad put the best of his efforts to design the toy. As a first module in the design, he intended to write a program that would reverse a numeric input given to it. 



He needs your help to write a function for reversing the digits of the given number N. Please assist him in the task.



Function Specifications:

int reverse(int n)

This function should return the reverse of an N-digit number.

Input format :
The input consists of an integer N.

Output format :
The output prints the reverse of the N-digit number.

Sample test cases :
Input 1 :
1234
Output 1 :
4321
Input 2 :
323333333
Output 2 :
333333323
Whitelist
Set 1:
